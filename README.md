# Campero Desktop - WPF MySQL Inventory Sales System

**Estado: Completo con Proveedores y Compras**

## Funcionalidades

- Login con roles
- Dashboard resumen
- CRUD Productos/Categorias
- CRUD Clientes
- Notas Venta transaccionales
- Inventario por almacén
- CRUD Usuarios
- Reportes PDF/Excel (5 tipos)
- CRUD Proveedores **NUEVO**
- Órdenes Compra por almacén **NUEVO**

## Ejecución

1. Visual Studio: Abrir CamperoDesktop.csproj
2. Ejecutar inventario_ventas.sql MySQL
3. App.config connection string
4. F5
